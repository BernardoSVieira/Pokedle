🎮 Pokedle
Um jogo de adivinhação inspirado no fenômeno Wordle, mas com o universo Pokémon! O desafio é descobrir qual é o Pokémon do dia utilizando dicas baseadas em tipos, gerações e atributos, consumindo dados reais da PokeAPI.

🚀 Tecnologias
Este projeto foi construído com as seguintes ferramentas modernas:

— Biblioteca para construção da interface.

— Build tool extremamente rápida para o desenvolvimento.

— Lógica principal do jogo.

— Estilização e responsividade.

🕹️ Como Jogar
Chute um Pokémon: Digite o nome de um Pokémon para começar.

Analise as cores:

🟩 Verde: O atributo está correto.

🟨 Amarelo: O atributo está parcialmente correto (ex: tipo certo, mas na posição errada).

🟥 Vermelho: O atributo está incorreto.

Acerte e vença: Use as pistas para encontrar o Pokémon secreto do dia!
